## Install

linux:
    * Install conan
    * mkdir build && cd build
    * conan remote add bincrafters https://api.bintray.com/conan/bincrafters/public-conan
    * conan install .. --build=missing

macos:
    brew install libomp
    brew install glfw3
    to install glew (http://macappstore.org/glew/)
